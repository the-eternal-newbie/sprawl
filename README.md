# sprawl

> A modern portfolio operating system.

## Packages

- `packages/mona-lisa-overdrive` — design system
- `packages/neuromancer` — window manager and multi-runtime app loader
- `packages/mnemonic-manifest` — App Manifest schema
- `apps/burning-chrome` — the frontend OS shell

## Sibling repositories (not in this monorepo)

- [`count-zero`](https://github.com/the-eternal-newbie/count-zero) — Django backend
- `sprawl-app-*` — individual mounted apps, one repo per language

## Development

```bash
pnpm install
pnpm dev
```

See `docs/` for architecture and contribution guides.
